SAP-native solver pipeline.

This solver wires together the SAP runtime components:

  1. SapFreeMotion.compute() computes SAP-order free motion.

  2. SapContactJacobian.compute() computes contact Jacobians and env-local dynamics matrices from SAP runtime data.

  3. SapContactSolve.solve() solves the SAP stage2 problem in SAP-order generalized velocities.

  4. SAP-native integration writes state_out.
